Разработка своего AI:
Для использования своего поместите модуль с ним в папку players. Реализация AI представляет из себя функцию, возвращающую строки команд в приложение. Команды полностью аналогичны пользовательским, но использовать команды типа "s" (вывод информации по компании) нет смысла, т.к. доступ к этим данным можно получить через интерфейсы классов. Их можно посмотреть в коде.
Общая структура вложенности такая:
функцияAI <-(состоит в классе) Director <- Company <- World
При этом в класс Director можно передать словарь произвольных параметров, которые будет использоваться в функции.

Далее, если Вы используете дополнительные параметры для Вашего AI, Вы можете подобрать оптимальные значения для них. Это делается с помощью функции calc из модуля calculation.
Она работает следующим образом: Вы передаете ей словарь параметров, которые используются в AI (parameters), в качестве значений этого словаря используется пара (tuple) из 2 чисел - наименьшее возможное значение этого параметра и наибольшее значение. Затем функция перебирает все возможные значения всех параметров и для каждой комбинации параметров запускается n-ое количество симуляций мира (которое также передается в функцию). Затем для каждой комбинации параметров выводится результат: сколько игр из скольки закончилось победой. Данная информация сохраняется в файл lastresults.log, так что Вы сможете просмотреть этот файл позже. ВНИМАНИЕ: этот файл перезаписывается повторным запуском.

Скорректируйте вызов этой функции в модуле main. Ей необходимо передать следующие аргументы:
* player - указатель на функцию, реализующую искуственный интеллект,
* nruns - количество запусков мира для каждой комбинации параметров,
* parameters - словарь определяющий пределы значений параметров. Вы можете заполнить его подходящими значениями из предыдущего шага (из файла lastresults.log), для подбора желаемой частоты побед Вашего AI

Добавьте имя функции своего AI в список func_list, а также его название (произвольное) в список players_list, в модуле players/__init__.py для того, чтобы его можно было использовать для игры
